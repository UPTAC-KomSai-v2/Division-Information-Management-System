#Dvision Information Management System Project

## For Frontend

###Prerequisites

Before installing, make sure you have the following:

| Requirement | Command to Check | Recommended Version |
|--------------|------------------|----------------------|
| Node.js | `node -v` | ≥ 20 |
| npm | `npm -v` | Latest |
| Quasar CLI (if used) | `quasar -v` | Latest |
| Git | `git --version` | Latest |
| Sass | — | Installed automatically, or run `npm install -D sass` |

Download Node.js: [https://nodejs.org](https://nodejs.org)

###Project Setup

#### 1. Clone the Repository
```bash
git clone https://github.com/UPTAC-KomSai-v2/Division-Information-Management-System.git
cd Division-Information-Management-System.git
cd Frontend

```
#### 2. Install Dependencies
```bash
npm install
```

#### 3. (Optional) Install Quasar CLI
```bash
npm i -g @quasar/cli
```

#### 4. Run the Development Server
```bash
quasar dev
```

#### Build for Production
```bash
qusar build
```
